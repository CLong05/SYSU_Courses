# SYSU_Courses

## Introduction

本项目分享了本人及友友们在中山大学计算机学院计算机科学与技术专业的课程资料、项目、笔记、回忆或收集的期末考试资料等。希望对你们有启发或帮助~如果喜欢不妨点个star嘻嘻！

资料仅供参考，同学们可自由分享取阅（如若**转发请注明出处**！谢谢！）。
